# 2. Cómo usar cada color

## `#DED9A6` — Amarillo crema

Úsalo para el **Hero principal**.

Da una sensación cálida, cercana y no intimidante.

Ejemplo:

**Fondo:** `#DED9A6`
**Texto:** `#121212`
**Botón principal:** `#050505` con texto blanco

Este bloque puede decir:

> Mueve dinero digital como mandar un mensaje.

---

## `#A7D4DE` — Celeste

Úsalo para bloques de **claridad, confianza y explicación simple**.

Ideal para:

* QR
* usernames
* pagos rápidos
* historial
* comprobantes

Ejemplo de bloque:

> Paga con QR o username.

El celeste comunica “esto es fácil, tranquilo, seguro”.

---

## `#DEA6BC` — Rosado

Úsalo para bloques más humanos/sociales.

Ideal para:

* links de pago
* cobrar a amigos/clientes
* compartir por WhatsApp
* comunidad
* freelancers
* eventos

Ejemplo:

> Cobra con un link.

El rosado hace que Parmelia se sienta menos financiera y más cotidiana.

---

## `#121212` — Negro suave

Úsalo para texto principal y bloques oscuros.

Mejor que usar negro puro para todo, porque es menos duro visualmente.

---

## `#050505` — Negro fuerte

Úsalo para:

* botones principales
* CTA final
* secciones de alto contraste
* footer

Ejemplo:

> Empieza con tu primer link.

Fondo negro fuerte, texto blanco, botones con colores pastel.

---

## `#FFFFFF` — Blanco

Úsalo para respirar.

No todos los bloques tienen que ser de color. Puedes intercalar bloques blancos para que la landing no se sienta infantil o saturada.

# 5. Reglas visuales para que no se vea infantil

Como los colores son pasteles, hay un riesgo: que parezca una landing de juguete o muy “cute”. Para evitarlo:

Usa mucho `#121212` para texto fuerte.

Usa botones negros.

Usa bordes negros delgados en algunos componentes.

Usa sombras mínimas, no sombras exageradas.

Usa mockups limpios de teléfono.

Usa mucho espacio en blanco.

No uses demasiados íconos redondeados o caricaturescos.

No uses gradientes arcoíris.

No uses emojis dentro de la landing principal.

# 7. Tipografía

Shippori Antique puede funcionar bien, pero úsala con jerarquía clara.

## Hero title

Grande, pero no demasiado apretado:

```css
font-size: clamp(48px, 8vw, 96px);
line-height: 0.95;
letter-spacing: -0.04em;
```

Texto:

```css
font-size: 20px;
line-height: 1.45;
```

## Títulos de sección

```css
font-size: clamp(36px, 5vw, 72px);
line-height: 1;
```

## Body

```css
font-size: 18px;
line-height: 1.6;
```

Como Shippori Antique tiene mucha personalidad, no haría textos larguísimos. Frases cortas. Mucho aire.
